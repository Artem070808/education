Тема 2.1 Квадратные уравнения. Дискриминант

Дорогие читатели, сегодня мы разберём базовую тему 8-9 классов по алгебре - квадратные уравнения. ax² + bx + с = 0.

Сейчас мы разберём способы решений квадратных уравнений. (ПОЛНЫХ)

Для начала разберёмся, как в целом решаются такие уравнения

Как вы уже знаете, решить уравнение в математике означает найти значение переменной, при которой выражение правдоподобно.

С квадратными уравнениями то же самое, есть два основных способа для нахождения корней уравнения. Сегодня мы разберём первый.

(найти корень-значит найти значение переменной).

1. Формула дискриминанта [D]

По значению дискриминанта определяется количество корней в уравнении, а именно:

1. Если D больше 0, то корня 2

2. Если D равен 0, то корень 1

3. Если D меньше 0, то корней у уравнения нет

Формула D=b²-4*a*c

Возьмём за пример уравнение 3x²+7x−6=0 и решим его с помощью дискриминанта.

Коэффициенты: a=3 ; b=7; c=(-6). Подставляем значения в формулу: D=7²-4*3*(-6)=121. Дискриминант равен 121. D>0 => корня 2. Корень D=√121=11

Далее, после того как мы нашли значение дискриминанта, нам понадобится формула корней квадратного уравнения, которая выглядит так:

Подставим значения в формулу и найдём наши корни, x1 и x2

x1=-7+11/2*3=4/6=2/3

x2=-7-11/2*3=-18/6=-3

Если D=0, то всё проходит идентично, только формула корней меняется на -b/2*a
